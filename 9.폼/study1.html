홈페이지를 사용하는 사용자가 내용을 입력할 수 있도록 미리 제공하는 양식태그

label태그느 이름표태그로 사용하며 사용자가 입력하는 입력박스에 대한 이름을 달아줄 때 사용한다.

laber태그와 input태그를 서로 연결하기 위해서는 input태그의 id값을 label태그의 for속성에 그대로 적용한다. - 홈페이지의 사용 편의를 위해 자주 사용하는 패턴

label태그가 디자인적으로 보이지 않더라도 실제로 html태그 label태그는 구성해야하며 css에서 안보이게 처리하는 방법을 사용한다.

input type속성이 radio인 경우 select태그와 같이 제한된 내용을 입력하게끔 유도할 때 사용하며, radio버튼의 name속성 값을 동일하게 하면 여러개의 radio버튼 중 1개만 선택되게 할 수 있다.

input type="text" 태그의 경우는 짧은 한줄의 텍스트를 입력시킬 떄 사용하며, textarea태그는 장문의 텍스트를 입력시킬 때 사용한다.
